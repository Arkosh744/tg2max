# User Journey — tg2max

## Flow 1: ZIP-экспорт (основной)

```
Пользователь                           Бот
    |                                    |
    |--- /start ----------------------->|
    |<-- Приветствие + 2 способа --------|  [keyboardMain: Справка, Статус]
    |                                    |
    |--- Экспортирует чат в TG Desktop   |
    |--- Архивирует в ZIP                |
    |--- Отправляет ZIP --------------->|
    |<-- "Загружаю и распаковываю..." ---|
    |<-- Автопоиск чата Max -------------|  [inline: список чатов]
    |                                    |
    |--- Выбирает чат ----------------->|
    |<-- "Шаг 1/2: Какой контент?" -----|  [inline: Всё/Текст/Медиа]
    |--- Выбирает фильтр контента ----->|
    |<-- "Шаг 2/2: За какой период?" ---|  [inline: 3мес/6мес/Всё время]
    |--- Выбирает период --------------->|
    |<-- Предпросмотр (dry-run stats) ---|  [keyboard: Подтвердить/Предпросмотр/Отмена]
    |                                    |
    |--- "Подтвердить перенос" --------->|
    |<-- Прогресс-бар (edit msg/10s) ----|  [keyboard: Пауза/Статус/Отменить]
    |    ...обновляется каждые 10 сек... |
    |<-- Финальная статистика ------------|  [keyboardMain]
```

## Flow 4: Админ-панель

```
Админ                                  Web UI
    |                                    |
    |--- GET /admin/ ------------------->|
    |<-- Redirect /admin/login ----------|
    |--- Вводит пароль ----------------->|
    |<-- Dashboard ----------------------|
    |    ├── Статистика (users, migrations, sent)
    |    ├── Активная миграция (SSE live, 3s)
    |    ├── Графики (Chart.js, 30 дней)
    |    └── Последние миграции (SSE, 18s)
    |                                    |
    |--- /admin/migrations ------------->|
    |<-- Таблица с фильтрами ------------|
    |    ├── Колонка "Тип" (Clone/ZIP badge)
    |    ├── Источник → Назначение
    |    ├── Статус (цветные badges)
    |    └── Пагинация
    |                                    |
    |--- /admin/migrations/42 ---------->|
    |<-- Детали миграции ----------------|
    |    ├── Источник и назначение (Clone→Max/TG)
    |    ├── Канал-источник (для clone)
    |    ├── Хронология (старт, финиш, длительность)
    |    └── Ошибка (если есть)
```

## Env-переменные (обязательные)

| Переменная | Описание | Обязательна |
|-----------|----------|-------------|
| `TELEGRAM_TOKEN` | Токен TG бота | Да |
| `MAX_TOKEN` | Токен Max бота | Да |
| `DB_PATH` | Путь к SQLite БД | Рекомендуется |
| `TG_APP_ID` | API ID для MTProto (my.telegram.org) | Для /clone |
| `TG_APP_HASH` | API Hash для MTProto | Для /clone |
| `USERBOT_SESSION_KEY` | Hex-ключ 32 байта для AES шифрования | Для /clone |
| `ADMIN_ENABLED` | Включить веб-панель | Нет |
| `ADMIN_PASSWORD` | Пароль админки | Если admin_enabled |
| `ADMIN_SECRET` | HMAC secret для cookie | Авто-генерируется |
| `ALLOWED_USER_IDS` | Whitelist TG user IDs (через запятую) | Рекомендуется |
